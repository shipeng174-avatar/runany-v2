# RunAny_v2 手动测试指南

> 以下功能需要 GUI 交互、系统状态依赖或用户输入，无法通过自动化脚本验证，请逐一测试。

---

## 1. Everything 文件搜索

### 1.1 SDK 直接 API 调用
- **前置条件**: Everything.exe 已运行且索引完整
- **测试步骤**:
  1. 在 `RunAny.ini` 中写入一条无路径应用，如 `记事本|notepad.exe`
  2. 启动 RunAny_v2
  3. 观察托盘提示是否显示"调用Everything搜索应用全路径"
- **预期**: notepad.exe 被成功解析为 `C:\Windows\System32\notepad.exe`

### 1.2 Everything 搜索窗 — 正则搜索
- **测试步骤**:
  1. 在设置界面 Tab5 勾选"正则搜索"(如已添加配置项 `EvUseRegex=1`)
  2. 选中一段文字
  3. 按 Everything 搜索热键（默认未设置，需先配置 `EvKey`）
- **预期**: Everything 窗口打开并执行正则搜索

### 1.3 Everything 多条件搜索
- **测试步骤**:
  1. 选中多行文字（如两行不同文件名）
  2. 按 Everything 搜索热键
- **预期**: Everything 窗口打开，搜索框包含 `|` 分隔的多个搜索词

## 2. 菜单交互

### 2.1 菜单弹出与运行
- **测试步骤**:
  1. 按 `Win+`` ` 热键
  2. 点击菜单中任意项（如记事本）
- **预期**: 菜单正常弹出，点击后启动对应程序

### 2.2 "添加到此菜单"
- **测试步骤**:
  1. 在资源管理器中选中一个 `.exe` 或 `.lnk` 文件
  2. 按 `Win+`` ` 弹出菜单
  3. 点击顶部的 "➕ 添加到此菜单"
- **预期**: 提示"已添加 N 个项目到菜单"，对应内容写入 `RunAny.ini`

### 2.3 多功能菜单（Ctrl+Shift+点击）
- **测试步骤**:
  1. 弹出菜单
  2. 按住 `Ctrl+Shift` 点击一个程序菜单项
- **预期**: 弹出多功能子菜单（运行/编辑/目录/透明/置顶/管理员/最小化/最大化/隐藏/结束进程/复制路径等）

### 2.4 选中文字显示在菜单顶部
- **测试步骤**:
  1. 在任意文本中选中一段文字
  2. 按 `Win+`` ` 弹出菜单
- **预期**: 菜单顶部显示选中的文字内容（带勾选标记）

### 2.5 菜单防闪烁
- **测试步骤**:
  1. 刚启动 RunAny 时立即按热键
- **预期**: 菜单不会闪烁或报错，而是延迟到就绪后显示

## 3. 热键

### 3.1 菜单弹出热键
- **测试步骤**: 按 `Win+`` ` → 菜单弹出 → 正常
- **其他触发方式**:
  - 双击 Ctrl（默认开启）
  - 双击 Alt（需在设置中开启）
  - 双击 Win 键
  - Ctrl + 鼠标右键
  - Shift + 鼠标右键
  - 鼠标 X1 / X2 / 中键

### 3.2 独立插件热键
- **测试步骤**:
  1. 右键托盘图标
  2. 点击"暂停插件" / "挂起插件" / "关闭插件"
- **预期**: 所有已启用的插件执行对应操作

### 3.3 TreeIcon 分类目录自定义热键
- **前置条件**: 在 `RunAny.ini` 中为分类节点添加 Tab 热键，如:
  ```
  -常用(&App)	!a
  ```
- **测试步骤**: 按 `Alt+A`
- **预期**: 直接弹出"常用"分类的子菜单

### 3.4 应用热键（菜单项 Tab 热键）
- **前置条件**: 在菜单项中添加 Tab 热键，如:
  ```
  记事本(&N)	^!n|notepad.exe
  ```
- **测试步骤**: 按 `Ctrl+Alt+N`
- **预期**: 直接启动记事本

### 6.1 多行匹配
- **测试步骤**:
  1. 选中多行包含网址的文字
  2. 按菜单热键
- **预期**: 如果多行内容匹配了一键直达的正则，直接执行对应的运行命令

- **测试步骤**:
  1. 选中 `1+2*3=` 
  2. 按菜单热键（需绑定 OneKeyMenu）
- **预期**: 自动替换为 `1+2*3=7`

### 6.3 逐行计算输出
- **测试步骤**:
  1. 选中多行计算式:
     ```
     1+2
     3*4
     ```
  2. 按菜单热键
- **预期**: 显示 ToolTip 计算结果 `3` 和 `12`，并复制到剪贴板
