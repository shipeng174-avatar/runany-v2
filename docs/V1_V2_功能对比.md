# RunAny V1 功能流程图 & V1/V2 对比

## 一、V1 启动初始化流程

```mermaid
flowchart TD
    START([RunAny.ahk 启动]) --> GLOBAL[全局变量定义<br/>名称/版本/路径/插件目录]
    GLOBAL --> CONFIG[01. 配置初始化 Config_Set<br/>读取RunAnyConfig.ini]
    CONFIG --> MENUVAR[02. 自定义变量 Menu_Var_Set<br/>解析系统变量/环境变量/用户变量]
    MENUVAR --> ICON[03. 图标初始化 Icon_Set<br/>图标路径/尺寸配置]
    ICON --> ENV[04. 运行环境检测 Run_Exist<br/>INI存在性/Everything DLL/编码]
    ENV --> HOTKEY[05. 热键注册<br/>遍历HotKeyList动态注册<br/>IfWinNotActive DisableGUI]
    HOTKEY --> TRAY[06. 托盘菜单构建 Menu_Tray_Add]
    TRAY --> EXTICON[07. 后缀图标初始化<br/>扫描ExeIcon/WebIcon/MenuIcon目录]
    EXTICON --> PLUGIN[08. 插件读取 Plugins_Read<br/>扫描RunPlugins目录]
    PLUGIN --> PLUGINAUTO[09. 自动启动插件 AutoRun_Plugins]
    PLUGINAUTO --> PLUGINCOM[10. COM对象注册 Plugins_Object_Register]
    PLUGINCOM --> RUNCTRL[11. 启动规则读取 RunCtrl_Read]
    RUNCTRL --> MENUBUILD[13-16. 菜单创建<br/>解析INI → Menu_Read]
    MENUBUILD --> EVSEARCH[17. Everything搜索无路径应用<br/>按需/全磁盘模式]
    EVSEARCH --> FILTER[19. 菜单过滤<br/>隐藏失效项/短语/网址等]
    FILTER --> OPENSEXT[21. 内部关联后缀 Open_Ext_Set]
    OPENSEXT --> EXEICON[22. EXE图标异步加载]
    EXEICON --> BACKUP[26. 自动备份 RunABackup]
    BACKUP --> AUTORELOAD[自动重载定时器<br/>检测INI修改时间]
    AUTORELOAD --> READY([RunAny 就绪])
```

## 三、V1 菜单显示决策树

```mermaid
flowchart TD
    MENUSHOW[Menu_Show 入口] --> CHECKZZ{选中内容类型}
    
    CHECKZZ -->|无选中| DEFAULT[显示默认菜单<br/>+ 软件专属分类<br/>+ 公共分类]
    CHECKZZ -->|选中文件| FILE[按后缀显示关联子菜单<br/>+ 公共分类<br/>+ 添加到菜单]
    CHECKZZ -->|选中文本| TEXT[一键直达正则匹配]
    
    TEXT --> REGEX1{正则匹配:公式?}
    REGEX1 -->|是| CALC[JScript eval 计算]
    REGEX1 -->|否| REGEX2{正则匹配:文件?}
    REGEX2 -->|是| OPENFILE[打开文件/目录]
    REGEX2 -->|否| REGEX3{正则匹配:网址?}
    REGEX3 -->|是| OPENURL[打开网址]
    REGEX3 -->|否| REGEX4{正则匹配:磁力?}
    REGEX4 -->|是| MAGNET[磁力链接]
    REGEX4 -->|否| SHOWTEXT[显示文本菜单<br/>+ 软件专属菜单<br/>+ Web菜单]
    
    DEFAULT & FILE & CALC & OPENFILE & OPENURL & MAGNET & SHOWTEXT --> INJECT[Folder注入<br/>Explorer/TC/DOpus/XYplorer<br/>当前目录注入菜单底部]
    INJECT --> SHOW[Menu.Show 显示菜单]
```

## 八、V1 启动规则系统

```mermaid
flowchart TD
    RULEINIT[RunCtrl_Read<br/>读取规则组和规则列表] --> GROUPS[规则组列表]
    GROUPS --> LOOP[遍历每个规则组]
    LOOP --> CHECK{规则条件判断}
    CHECK -->|菜单变量| VARCHECK[对比变量值<br/>等于/不等于/大于/小于/正则]
    CHECK -->|内置函数| FUNCCHECK[调用内置检测函数<br/>电脑名/用户名/IP/WiFi等]
    CHECK -->|插件函数| PLUGCHECK[调用插件函数]
    
    CHECK -->|AND 全部成立| RUN[执行启动应用]
    CHECK -->|OR 任一成立| RUN
    CHECK -->|不满足| SKIP[跳过]
    
    RUN --> METHOD{运行方式}
    METHOD -->|菜单项| MENUITEM[通过菜单运行]
    METHOD -->|全路径| DIRECT[直接运行]
    METHOD -->|管理员| ADMIN[*RunAs运行]
    METHOD -->|定时循环| TIMER[设置定时器继续检测]
```

## 十、V2 需补全的关键功能（按优先级排序）

### P0 - 核心缺失
1. **Space/右键/中键/XButton 复用HoldKey编号** — V1的RunAny_Menu插件核心功能，菜单内按键执行HoldKey操作

### P1 - 重要功能
2. **JumpSearch 跳过批量搜索确认** — 配置项已有，运行逻辑未接入
3. **Everything管理员权限同步** — V1会自动以管理员重启Everything
4. **资源管理器快速切换(Ctrl+G)** — V1 L1485，在资源管理器中Ctrl+G弹出菜单切换当前目录

### P2 - 增强功能
5. **检查更新系统** — V1有完整的GitHub/Gitee自动更新
6. **注册表关联图标** — 未知后缀时从注册表查找默认图标
7. **EXE图标生成(ResourcesExtract)** — V1 Tab9中的批量图标提取
8. **外部调用接口** — Remote_Dyna_Run/Remote_Menu_Run

### P3 - 不复刻（用户指定排除）
- XiaoYao_plus 插件
- xiaoyao_防卡键 插件
